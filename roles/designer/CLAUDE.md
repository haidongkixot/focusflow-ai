# Designer Role — FocusFlow AI

## Design System
- Brand color: Rose `#f43f5e` (accent-400)
- Glow: `rgba(244, 63, 94, 0.3)`
- Background: `#0f0f1a` (bg-base), `#16162a` (bg-surface), `#1e1e35` (bg-elevated)
- Text: `#e5e7eb` (primary), `#9ca3af` (secondary), `#6b7280` (muted)
- Font: Inter (Google Fonts)

## Utility Classes
- `.btn-primary` — rose bg, shadow-glow
- `.btn-secondary` — outlined, border-gray-700
- `.card` — bg-surface, border-gray-800, rounded-2xl
- `.card-hover` — card + hover:border-accent-400/30 + hover:shadow-glow
- `.input-field` — bg-elevated, focus:border-accent-400

## Timer UI
- Circular SVG progress ring (radius 120, stroke 8)
- Large mono countdown font (text-6xl)
- Accent color per session type: Rose=Focus, Emerald=Short Break, Blue=Long Break

## Icons
lucide-react — Zap (brand), Timer, ListTodo, BarChart3, Brain, Trophy
